# Construction Lift Monitoring Dashboard

This project is a **React-based dashboard** that simulates a construction site lift monitoring system.

## Features

* Lift movement between floors
* Floor position visualization
* Lift control panel (Move Up / Move Down)
* Status monitoring (Running / Stopped / Emergency)
* Direction indicator
* Smooth lift animation
* Real-time height calculation

## Technologies Used

* React.js
* JavaScript
* HTML
* CSS

## How It Works

The dashboard simulates a construction lift where:

* Each floor represents **3 meters of height**
* The lift can move between **0–9 floors**
* The system updates **floor, height, direction, and status** dynamically

## Controls

* **Move Up** → Lift moves one floor up
* **Move Down** → Lift moves one floor down
* **Stop** → Lift status changes to stopped
* **Emergency** → Emergency stop simulation
